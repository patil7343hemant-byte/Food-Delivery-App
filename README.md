# Food Delivery App

A simple full-stack food delivery application built for internship submission.

## Features
- Restaurant listing
- Search by restaurant/cuisine
- Menu browsing
- Add to cart
- Checkout form
- Order placement
- Order tracking with status and ETA
- Recent orders dashboard

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Data: JSON file storage
- Database schema: `database.sql`

## Folder Structure
```text
Food-Delivery-App/
├── frontend/
├── backend/
├── screenshots/
├── report.pdf
├── README.md
└── database.sql
```

## Run Locally
1. Install Node.js
2. Open terminal in project folder
3. Run:
   ```bash
   npm install
   npm start
   ```
4. Open:
   `http://localhost:3000`
